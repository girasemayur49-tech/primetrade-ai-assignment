# PrimeTrade AI Assignment

##  Project Overview
This project is part of an AI/Data Science assignment.  
It includes data analysis and visualization of trading/market data using Python.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

##  Project Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of trading insights
- Profit analysis based on market sentiment

## 📁 Files Included
- notebook.ipynb (main analysis file)

##  How to Run
1. Open Jupyter Notebook or Google Colab  
2. Run the .ipynb file step by step  
3. View outputs and graphs
